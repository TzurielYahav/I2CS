## Introduction to Computer Science Projects
This is a repository for projects for the course - Introduction to Computer Science in Java (Ariel University, 2026 - fall semester - A). This repository contains the code for all of the assignments for the CS-101 course.

## Usage
git clone https://github.com/TzurielYahav/I2CS.git
npm install
